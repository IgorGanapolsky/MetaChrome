Expo Go Connection Instructions
===============================

Your Expo server is running with tunnel mode enabled.

To connect from your phone:

1. Install Expo Go app:
   - iOS: https://apps.apple.com/app/expo-go/id982107779  
   - Android: https://play.google.com/store/apps/details?id=host.exp.exponent

2. Run this command to see the QR code:
   cd /workspace/frontend && npx expo start --tunnel

3. The QR code will appear in the terminal output

4. Open Expo Go app on your phone and scan the QR code

Alternative: If you see a URL like "exp://..." in the terminal, you can:
- Open Expo Go app
- Tap "Enter URL manually"  
- Paste the exp:// URL

The server is running on port 8081 with tunnel mode active.
